# Implementation plans

Issue-scoped delivery plans live in `docs/implementation-plans/` at the repo root, one Markdown file per GitHub issue.
File names follow `<issue-number>-<slug>.md`.

Browse the directory on GitHub: [epfl-enac/co2-calculator/tree/main/docs/implementation-plans](https://github.com/epfl-enac/co2-calculator/tree/main/docs/implementation-plans).

These plans are not rendered into the MkDocs site; they stay as raw Markdown so LLM agents and humans can grep them directly in the repo.
